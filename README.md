# 🏎️ Formula-1 Data Analysis

A Formula-1 Data Analytics and Winner Prediction web app built using **Python**, **Streamlit**, and **Machine Learning**.

## 🚀 Features

* Interactive F1 Dashboard
* Driver & Constructor Analysis
* Race Trend Visualizations
* Win Prediction using ML

## 🛠️ Tech Stack

* Python
* Streamlit
* Pandas
* Plotly
* Scikit-learn
* Matplotlib
* Seaborn

## 📂 Dataset

The project uses historical Formula-1 datasets:

* Drivers
* Constructors
* Circuits
* Races
* Results
* Qualifying

## ▶️ Run Locally

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```

## 🤖 ML Model

* Random Forest Classifier
* Accuracy: ~95%
